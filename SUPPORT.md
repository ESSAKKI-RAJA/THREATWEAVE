# Support

## How to Get Help

THREATWEAVE is an actively maintained open-source project. We offer several channels for getting help, depending on the nature of your issue.

---

## Before You Ask

Please check these resources first — your question may already be answered:

- 📖 [README.md](README.md) — Setup, configuration, and deployment
- 🔧 [Troubleshooting section](README.md#troubleshooting) — Common issues and solutions
- 🔍 [Existing GitHub Issues](https://github.com/ESSAKKI-RAJA/THREATWEAVE/issues) — Search for similar problems
- 💬 [GitHub Discussions](https://github.com/ESSAKKI-RAJA/THREATWEAVE/discussions) — Community Q&A

---

## Support Channels

### GitHub Issues — Bug Reports
**Use for:** Reproducible bugs, unexpected behavior, broken functionality

1. Go to [Issues → New Issue](https://github.com/ESSAKKI-RAJA/THREATWEAVE/issues/new/choose)
2. Select **Bug Report** template
3. Fill in the template completely, including:
   - Steps to reproduce
   - Expected vs actual behavior
   - Environment details (OS, Node version, browser)
   - Relevant logs or screenshots

Issues without sufficient detail may be closed with a request for more information.

### GitHub Discussions — Questions & Ideas
**Use for:** General questions, architecture discussions, feature brainstorming, "how do I..." questions

→ [GitHub Discussions](https://github.com/ESSAKKI-RAJA/THREATWEAVE/discussions)

### GitHub Issues — Feature Requests
**Use for:** Requesting new functionality or improvements

1. Go to [Issues → New Issue](https://github.com/ESSAKKI-RAJA/THREATWEAVE/issues/new/choose)
2. Select **Feature Request** template
3. Describe the problem you're solving (not just the solution)

### Security Vulnerabilities
**Do NOT** open a public issue for security vulnerabilities.

See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## Response Time Expectations

| Channel | Expected Response |
|---|---|
| Security vulnerabilities | 48 hours |
| Bug reports (critical) | 24–48 hours |
| Bug reports (non-critical) | 3–5 business days |
| Feature requests | 1–2 weeks |
| General questions (Discussions) | Community-driven |
